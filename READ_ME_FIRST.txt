🎯 WHAT THE KPIs MEAN (QUICK REFERENCE):
═══════════════════════════════════════════════════════════════

After simulation completes, you see 5 boxes:

1. Total P&L
   → Your total profit/loss
   → Green if positive, red if negative
   → Good: > $1,000

2. Sharpe Ratio
   → Risk-adjusted return
   → Good: > 1.0
   → Excellent: > 1.2

3. Max Drawdown
   → Biggest loss from peak
   → Good: < 10%
   → Warning: > 15%

4. Win Rate
   → % profitable trades
   → Good: > 55%
   → Random: ~50%

5. Avg Inventory
   → Average position size
   → Good: < 200
   → High risk: > 400

🎯 STEP 1: PREPARE PROJECT (30 seconds)
═══════════════════════════════════════════════════════════════

Run this command:

  bash RENDER_QUICK_DEPLOY.sh

This creates render.yaml and prepares everything.

═══════════════════════════════════════════════════════════════

🎯 STEP 2: CREATE GITHUB REPO (2 minutes)
═══════════════════════════════════════════════════════════════

A. Go to: https://github.com/new

B. Fill in:
   • Repository name: adaptive-liquidity-provision
   • Visibility: Public ✅ (required for free tier)
   • Click "Create repository"

C. Push your code (run these commands):

   git remote add origin https://github.com/YOUR_USERNAME/adaptive-liquidity-provision.git
   git branch -M main
   git push -u origin main

   (Replace YOUR_USERNAME with your GitHub username)

═══════════════════════════════════════════════════════════════

🎯 STEP 3: CREATE RENDER ACCOUNT (1 minute)
═══════════════════════════════════════════════════════════════

A. Go to: https://render.com/

B. Click "Get Started"

C. Sign up with GitHub (easiest option)

═══════════════════════════════════════════════════════════════

🎯 STEP 4: DEPLOY ON RENDER (2 minutes)
═══════════════════════════════════════════════════════════════

A. Go to: https://dashboard.render.com/

B. Click "New +" button → "Web Service"

C. Click "Connect GitHub"

D. Find and select: adaptive-liquidity-provision

E. Render will auto-detect render.yaml ✅

F. Click "Create Web Service"

═══════════════════════════════════════════════════════════════

🎯 STEP 5: WAIT FOR BUILD (3-5 minutes)
═══════════════════════════════════════════════════════════════

Watch the build logs. You'll see:

  ✅ Cloning repository...
  ✅ Installing dependencies...
  ✅ Build successful!
  ✅ Starting service...
  ✅ Live at: https://adaptive-liquidity-provision.onrender.com

═══════════════════════════════════════════════════════════════

🎉 DONE! YOUR DASHBOARD IS LIVE!
═══════════════════════════════════════════════════════════════

Your URL:
  https://adaptive-liquidity-provision.onrender.com

Or:
  https://YOUR-SERVICE-NAME.onrender.com
